# <a name="mixed-reality-documentation"></a>混合现实文档

<h2>HoloLens 2 简介</h2>

[![HoloLens 2](images/hololens2.jpg)](https://www.microsoft.com/hololens/hardware)

[HoloLens 2](https://www.microsoft.com/hololens/hardware) 提供最舒适的沉浸式混合现实体验，其行业领先的解决方案可以在数分钟内为你带来价值 - 这一切由 Microsoft 推出的云和 AI 服务所特有的可靠性、安全性和可伸缩性提供坚实支持。
