# <a name="keyboard"></a>键盘

![键盘](images/UX/UX_Hero_Keyboard.jpg)

混合现实应用程序可以随时调用系统键盘。 请注意，系统键盘的行为取决于目标平台的功能，例如，HoloLens 2 上的键盘支持直接交互，而 HoloLens 上的键盘支持 "注视" 和 "空中点击" 笔势输入。
