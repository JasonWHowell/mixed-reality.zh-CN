# <a name="advanced-hololens-emulator-and-mixed-reality-simulator-input"></a>高级的 HoloLens 仿真程序和混合现实模拟器输入

大多数仿真程序用户只需使用有关的基本输入的控件[HoloLens 模拟器](using-the-hololens-emulator.md#basic-emulator-input)或[Windows Mixed Reality 模拟器](using-the-windows-mixed-reality-simulator.md#basic-simulator-input)。 下面的详细信息适用于高级用户发现需要模拟更复杂类型的输入。


## <a name="concepts"></a>概念

若要开始控制虚拟输入到 HoloLens 仿真程序和 Windows Mixed Reality 模拟器，你首先应了解几个概念。

动作是指控制和更改的位置和方向的场景中的某些内容。 对于目标可控制对象，动作控制使用旋转和平移 （运动） 沿三个轴。
* **偏航角**:打开左侧或右侧。
* **间距**:向上或向下打开。
* **回滚**:将鼠标移到并行。
* **X**:左或向右移动。
* **Y**:上移或下移。
* **Z**:向前或向后移动。

[手势](gestures.md)和动作控制器输入紧密映射到如何在物理设备：
* **操作**:这模拟按下 thumb 到食指或拉取操作按钮在控制器上的操作。 例如，操作输入可用于模拟-敲击手势，滚动到的内容，再按下并保持到。
* **[布隆](gestures.md#bloom)/System 手势或家庭**:若要返回到命令行程序以及执行系统操作使用 HoloLens 布隆/系统笔势或控制器的主页按钮。

手具有丰富 reprresentation HoloLens 2 中。  除了跟踪/未被跟踪，并可用于推动手势外，手现在具有一个明确的主干模型调整到其大小和公开给开发人员。  这带来了 26 跟踪的点上每个指针。  
* **联合**:一个给定跟踪手的形状的 20 个跟踪位置。 这将产生一个点是与之关联的 3d 空间。
* **会带来**:所有跟踪手的形状的连接的完整集合。 在此期间，这是 26 关节的集合。 

在此期间，我们不会公开直接控制的每个联合位置单独通过仿真程序用户界面，但可以通过模拟 API 设置。 相反，我们有一组有用的代表性带来，仿真程序可用于切换。

此外可以控制模拟的传感器输入状态：
* **重置**:这将为其默认值来返回所有模拟的传感器。  从开始 HoloLens 2 仿真程序，重置的范围可以为一个或两个指针的具有吸引力的所需的 hand(s) 使用适当的修改键或按钮 （左和/或右 alt 键或在游戏板上的左侧和/或右侧缓冲器）。
* **跟踪**:循环浏览位置跟踪模式。 这包括：
  * **默认**：操作系统选择最佳的跟踪模式基于对系统进行的请求。
   * **方向**:强制仅限方向的跟踪，而不考虑对系统进行的请求。
   * **位置**:强制位置跟踪，而不考虑对系统进行的请求。

## <a name="targeting"></a>定位 

某些上述输入概念独立存在其自身。  操作、 布隆/系统、 重置和跟踪是完整的概念、 不需要和不受影响的面向任何其他修饰符。  但是，剩余的概念可以应用于多个目标之一。 我们引入了方法来指定哪些适用命令应该应用到的目标。  在所有情况下，则可以通过 UI 或通过键盘，按下，该对象传递给 targtet 指定。  在某些情况下，还有可能直接指定与 xbox 控制器。

## <a name="controlling-an-app"></a>控制应用

对于日常使用，建议使用以下一组控件：

|  操作 |  键盘和鼠标 |  控制器 | 
|----------|----------|----------|
|  正文 X |  A / D |  左的摇杆左 / 右 | 
|  正文 Y |  向上翻页/向下翻页 |  DPad 向上 / 向下 | 
|  正文 Z |  W / S |  左的摇杆向上 / 向下 | 
|  正文偏航 |  拖动鼠标左 / 右 |  右摇杆左 / 右 | 
|  Head 偏航角 |  H 并拖动鼠标向左 / 右 |  （在键盘） H + 右摇杆左 / 右 | 
|  Head 间距 |  向上 / 向下拖动鼠标 |  右摇杆向上 / 向下 | 
|  Head 回滚 |  Q / E |  DPad 左 / 右 | 
|  手动/控制器 X |  Alt + A / D |  即时权限提升 + 左的摇杆左 / 右 | 
|  手动/控制器 Y |  Alt + Page up/向下翻页 |  即时权限提升 + DPad 向上 / 向下 | 
|  手动/控制器 Z |  Alt + W / S |  即时权限提升 + 左的摇杆向上 / 向下 | 
|  手动/控制器偏航 |  按住 alt 并拖动鼠标向左 / 右 |  即时权限提升 + 右摇杆左 / 右 | 
|  手动/控制器间距 |  按住 alt 并拖动鼠标向上 / 向下 |  即时权限提升 + 右摇杆向上 / 向下 | 
|  手动/控制器前 |  Alt + Q / E |  即时权限提升 + DPad 左 / 右 | 
|  操作 |  鼠标右按钮 |  触发器 | 
|  布隆 / 系统 / 主页 |  F2 或 Windows 键 |  B 按钮 | 
|  Reset |  退出 |  “开始”按钮 | 
|  跟踪 |  T |  X 按钮 | 
|  滚动 |  Alt + 向右键鼠标按钮 + 向上 / 向下拖动鼠标 |  即时权限提升 + 触发器 + 右摇杆向上 / 向下 | 
|  移动/旋转速度更快 | 左或向右 Shift 键 | 按下并保持正确摇杆 |
|  移动/旋转速度缓慢 | 左或向右 Ctrl 键 | 按下并保持左的摇杆 |

## <a name="perception-simulation-control-panel-keyboard-shortcuts"></a>Perception 模拟控制面板键盘快捷方式

以下键盘快捷方式可用于访问 Perception 模拟控制面板和启用或禁用 PC 输入的设备的使用模拟。

| 操作 | 快捷方式 | 说明/说明 |
|-----------|----------|-------------|
| 切换用于模拟键盘 | F4 | 关闭功能时，键盘输入转至的 HoloLens 或 Windows Mixed Reality 应用程序。 |
| 切换用于模拟鼠标 | F5 | 鼠标输入时关闭，请转到混合现实环境 (仅 Windows Mixed Reality) |
| 切换用于模拟游戏板 | F6 | 通过模拟关闭功能时，将忽略游戏板输入 |
| 显示或隐藏控件面板 | F7 | |
| 将键盘焦点设置到控制面板 | F8 | 如果面板不当前可见，则它将会显示第一个。 |
| 停靠或取消停靠面板向/从仿真程序或混合现实门户窗口 | F9 | 如果窗口已关闭时取消停靠，它是停靠和隐藏。 |
