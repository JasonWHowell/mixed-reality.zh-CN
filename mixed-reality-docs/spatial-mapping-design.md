## <a name="the-environment-scanning-experience"></a><span data-ttu-id="332ef-124">环境扫描体验</span><span class="sxs-lookup"><span data-stu-id="332ef-124">The environment scanning experience</span></span>

<span data-ttu-id="332ef-125">使用空间映射的每个应用程序都应考虑提供 "扫描体验";应用程序引导用户扫描应用程序正常运行所必需的表面所使用的过程。</span><span class="sxs-lookup"><span data-stu-id="332ef-125">Each application that uses spatial mapping should consider providing a 'scanning experience'; the process through which the application guides the user to scan surfaces that are necessary for the application to function correctly.</span></span>

<span data-ttu-id="332ef-126">![扫描示例](images/sr-mixedworld-140429-8pm-00068-1000px.png)</span><span class="sxs-lookup"><span data-stu-id="332ef-126">![Example of scanning](images/sr-mixedworld-140429-8pm-00068-1000px.png)</span></span><br>
<span data-ttu-id="332ef-127">*扫描示例*</span><span class="sxs-lookup"><span data-stu-id="332ef-127">*Example of scanning*</span></span>

<span data-ttu-id="332ef-128">此扫描体验的性质可能因每个应用程序的需求而有很大差别, 但两个主要原则应指导其设计。</span><span class="sxs-lookup"><span data-stu-id="332ef-128">The nature of this scanning experience can vary greatly depending upon each application's needs, but two main principles should guide its design.</span></span>

<span data-ttu-id="332ef-129">首先,**清除与用户的通信是主要关注**点。</span><span class="sxs-lookup"><span data-stu-id="332ef-129">Firstly, **clear communication with the user is the primary concern**.</span></span> <span data-ttu-id="332ef-130">用户应始终知道是否满足应用程序的要求。</span><span class="sxs-lookup"><span data-stu-id="332ef-130">The user should always be aware of whether the application's requirements are being met.</span></span> <span data-ttu-id="332ef-131">如果不满足这些要求, 则应该立即清楚地说明这种情况的原因, 并且应迅速采取适当的措施。</span><span class="sxs-lookup"><span data-stu-id="332ef-131">When they are not being met, it should be immediately clear to the user why this is so and they should be quickly led to take the appropriate action.</span></span>

<span data-ttu-id="332ef-132">其次,**应用程序应尝试在效率和可靠性之间取得平衡**。</span><span class="sxs-lookup"><span data-stu-id="332ef-132">Secondly, **applications should attempt to strike a balance between efficiency and reliability**.</span></span> <span data-ttu-id="332ef-133">如果可以**可靠地**执行此操作, 则应用程序应自动分析空间映射数据以节省用户时间。</span><span class="sxs-lookup"><span data-stu-id="332ef-133">When it is possible to do so **reliably**, applications should automatically analyze spatial mapping data to save the user time.</span></span> <span data-ttu-id="332ef-134">如果无法可靠地执行此操作, 则应用程序应改为允许用户使用其所需的其他信息快速提供应用程序。</span><span class="sxs-lookup"><span data-stu-id="332ef-134">When it is not possible to do so reliably, applications should instead enable the user to quickly provide the application with the additional information it requires.</span></span>

<span data-ttu-id="332ef-135">若要帮助设计正确的扫描体验, 请考虑以下哪些可能适用于你的应用程序:</span><span class="sxs-lookup"><span data-stu-id="332ef-135">To help design the right scanning experience, consider which of the following possibilities are applicable to your application:</span></span>

* <span data-ttu-id="332ef-136">**无扫描体验**</span><span class="sxs-lookup"><span data-stu-id="332ef-136">**No scanning experience**</span></span>
   * <span data-ttu-id="332ef-137">如果没有任何引导式扫描体验, 应用程序可以正常工作;它将了解在自然用户移动过程中观察到的图面。</span><span class="sxs-lookup"><span data-stu-id="332ef-137">An application may function perfectly without any guided scanning experience; it will learn about surfaces that are observed in the course of natural user movement.</span></span>
   * <span data-ttu-id="332ef-138">例如, 允许用户在带有全息喷涂画图的表面上绘图的应用程序只需要知道当前对用户可见的表面。</span><span class="sxs-lookup"><span data-stu-id="332ef-138">For example an application that lets the user draw on surfaces with holographic spray paint requires knowledge only of the surfaces currently visible to the user.</span></span>
   * <span data-ttu-id="332ef-139">如果环境中的用户已使用 HoloLens 花费了大量时间, 则可能会完全扫描该环境。</span><span class="sxs-lookup"><span data-stu-id="332ef-139">The environment may be completely scanned already if it is one in which the user has already spent a lot of time using the HoloLens.</span></span>
   * <span data-ttu-id="332ef-140">请记住, 空间映射使用的相机只能在用户前面看到 3.1 m, 因此, 空间映射将不知道任何更远的图面, 除非用户在过去从更远的距离观察到它们。</span><span class="sxs-lookup"><span data-stu-id="332ef-140">Bear in mind however that the camera used by spatial mapping can only see 3.1m in front of the user, so spatial mapping will not know about any more distant surfaces unless the user has observed them from a closer distance in the past.</span></span>
   * <span data-ttu-id="332ef-141">因此, 用户了解已经扫描了哪些表面, 应用程序应提供此效果的视觉反馈, 例如, 将虚拟阴影转换到扫描图面上可能会帮助用户在这些表面上放置全息影像。</span><span class="sxs-lookup"><span data-stu-id="332ef-141">So the user understands which surfaces have been scanned, the application should provide visual feedback to this effect, for example casting virtual shadows onto scanned surfaces may help the user place holograms on those surfaces.</span></span>
   * <span data-ttu-id="332ef-142">对于这种情况, 应将空间图面观察器的边界卷更新为主体锁定的[空间坐标系统](coordinate-systems.md), 使其跟随用户。</span><span class="sxs-lookup"><span data-stu-id="332ef-142">For this case, the spatial surface observer's bounding volumes should be updated each frame to a body-locked [spatial coordinate system](coordinate-systems.md), so that they follow the user.</span></span>

* <span data-ttu-id="332ef-143">**查找合适的位置**</span><span class="sxs-lookup"><span data-stu-id="332ef-143">**Find a suitable location**</span></span>
   * <span data-ttu-id="332ef-144">应用程序可设计为在具有特定要求的位置使用。</span><span class="sxs-lookup"><span data-stu-id="332ef-144">An application may be designed for use in a location with specific requirements.</span></span>
   * <span data-ttu-id="332ef-145">例如, 应用程序可能需要用户附近的空白区域, 以便他们能够安全地练习全息 kung-fu。</span><span class="sxs-lookup"><span data-stu-id="332ef-145">For example, the application may require an empty area around the user so they can safely practice holographic kung-fu.</span></span>
   * <span data-ttu-id="332ef-146">应用程序应提前向用户传达任何特定要求, 并通过清晰的视觉反馈来强化。</span><span class="sxs-lookup"><span data-stu-id="332ef-146">Applications should communicate any specific requirements to the user up-front, and reinforce them with clear visual feedback.</span></span>
   * <span data-ttu-id="332ef-147">在此示例中, 应用程序应可视化所需空区域的范围, 并以可视方式突出显示此区域内任何不需要的对象的状态。</span><span class="sxs-lookup"><span data-stu-id="332ef-147">In this example, the application should visualize the extent of the required empty area and visually highlight the presence of any undesired objects within this zone.</span></span>
   * <span data-ttu-id="332ef-148">对于这种情况, 空间图面观察器的边界卷应在所选位置使用世界锁定的[空间坐标系统](coordinate-systems.md)。</span><span class="sxs-lookup"><span data-stu-id="332ef-148">For this case, the spatial surface observer's bounding volumes should use a world-locked [spatial coordinate system](coordinate-systems.md) in the chosen location.</span></span>

* <span data-ttu-id="332ef-149">**查找图面的合适配置**</span><span class="sxs-lookup"><span data-stu-id="332ef-149">**Find a suitable configuration of surfaces**</span></span>
   * <span data-ttu-id="332ef-150">应用程序可能需要特定的图面配置, 例如两个大型、平整、相反的壁, 以创建全息的镜像厅。</span><span class="sxs-lookup"><span data-stu-id="332ef-150">An application may require a specific configuration of surfaces, for example two large, flat, opposing walls to create a holographic hall of mirrors.</span></span>
   * <span data-ttu-id="332ef-151">在这种情况下, 应用程序需要对空间映射提供的图面进行分析以检测适当的图面, 并将用户定向到它们。</span><span class="sxs-lookup"><span data-stu-id="332ef-151">In such cases the application will need to analyze the surfaces provided by spatial mapping to detect suitable surfaces, and direct the user toward them.</span></span>
   * <span data-ttu-id="332ef-152">如果应用程序的表面分析不完全可靠, 用户应具有回退选项。</span><span class="sxs-lookup"><span data-stu-id="332ef-152">The user should have a fallback option if the application's surface analysis is not completely reliable.</span></span> <span data-ttu-id="332ef-153">例如, 如果应用程序错误地将门口标识为平整壁, 则用户需要一种简单的方法来更正此错误。</span><span class="sxs-lookup"><span data-stu-id="332ef-153">For example, if the application incorrectly identifies a doorway as a flat wall, the user needs a simple way to correct this error.</span></span>

* <span data-ttu-id="332ef-154">**扫描部分环境**</span><span class="sxs-lookup"><span data-stu-id="332ef-154">**Scan part of the environment**</span></span>
   * <span data-ttu-id="332ef-155">应用程序可能希望只捕获用户所指示的部分环境。</span><span class="sxs-lookup"><span data-stu-id="332ef-155">An application may wish to only capture part of the environment, as directed by the user.</span></span>
   * <span data-ttu-id="332ef-156">例如, 应用程序会扫描房间的某个部分, 以使用户可以为想要销售的家具发布全息保密广告。</span><span class="sxs-lookup"><span data-stu-id="332ef-156">For example, the application scans part of a room so the user may post a holographic classified ad for furniture they wish to sell.</span></span>
   * <span data-ttu-id="332ef-157">在这种情况下, 应用程序应该捕获用户在扫描期间观察到的区域内的空间映射数据。</span><span class="sxs-lookup"><span data-stu-id="332ef-157">In this case, the application should capture spatial mapping data within the regions observed by the user during their scan.</span></span>

* <span data-ttu-id="332ef-158">**扫描整个聊天室**</span><span class="sxs-lookup"><span data-stu-id="332ef-158">**Scan the whole room**</span></span>
   * <span data-ttu-id="332ef-159">应用程序可能需要扫描当前房间中的所有表面, 包括用户之后的所有表面。</span><span class="sxs-lookup"><span data-stu-id="332ef-159">An application may require a scan of all of the surfaces in the current room, including those behind the user.</span></span>
   * <span data-ttu-id="332ef-160">例如, 游戏可能将用户作为 Gulliver 的角色, 在 siege 中, 从数百个微小 Lilliputians 接近所有方向。</span><span class="sxs-lookup"><span data-stu-id="332ef-160">For example, a game may put the user in the role of Gulliver, under siege from hundreds of tiny Lilliputians approaching from all directions.</span></span>
   * <span data-ttu-id="332ef-161">在这种情况下, 应用程序需要确定当前房间中已经扫描了多少个面, 并指示用户的注视填充了重大间隙。</span><span class="sxs-lookup"><span data-stu-id="332ef-161">In such cases, the application will need to determine how many of the surfaces in the current room have already been scanned, and direct the user's gaze to fill in significant gaps.</span></span>

* <span data-ttu-id="332ef-164">**获取环境的初始快照**</span><span class="sxs-lookup"><span data-stu-id="332ef-164">**Take an initial snapshot of the environment**</span></span>

* <span data-ttu-id="332ef-170">**采用用户启动的环境快照**</span><span class="sxs-lookup"><span data-stu-id="332ef-170">**Take user-initiated snapshots of the environment**</span></span>

   * <span data-ttu-id="332ef-172">例如, 用户可以通过在不同时刻捕获其姿势来创建多个 3D "statues"。</span><span class="sxs-lookup"><span data-stu-id="332ef-172">For example, the user could create multiple 3D 'statues' of a friend by capturing their poses at different moments.</span></span>

* <span data-ttu-id="332ef-173">**允许用户更改环境**</span><span class="sxs-lookup"><span data-stu-id="332ef-173">**Allow the user to change the environment**</span></span>
   * <span data-ttu-id="332ef-174">应用程序可用于实时响应用户环境中所做的任何更改。</span><span class="sxs-lookup"><span data-stu-id="332ef-174">An application may be designed to respond in real-time to any changes made in the user's environment.</span></span>
   * <span data-ttu-id="332ef-175">例如, 如果在另一侧发生全息部游戏, 则用户绘制窗帘可能会触发 "场景变化"。</span><span class="sxs-lookup"><span data-stu-id="332ef-175">For example, the user drawing a curtain could trigger 'scene change' for a holographic play taking place on the other side.</span></span>

* <span data-ttu-id="332ef-176">**指导用户避免空间映射数据中的错误**</span><span class="sxs-lookup"><span data-stu-id="332ef-176">**Guide the user to avoid errors in the spatial mapping data**</span></span>

## <a name="mesh-processing"></a><span data-ttu-id="332ef-183">网格处理</span><span class="sxs-lookup"><span data-stu-id="332ef-183">Mesh processing</span></span>

<span data-ttu-id="332ef-184">它可能有助于检测图面中的常见错误类型, 并根据需要对空间映射数据进行筛选、删除或修改。</span><span class="sxs-lookup"><span data-stu-id="332ef-184">It may help to detect common types of errors in surfaces and to filter, remove or modify the spatial mapping data as appropriate.</span></span>

<span data-ttu-id="332ef-185">请记住, 空间映射数据应尽可能忠实到实际的表面, 因此, 任何正在进行的处理都将进一步从 "真实" 改变你的曲面。</span><span class="sxs-lookup"><span data-stu-id="332ef-185">Bear in mind that spatial mapping data is intended to be as faithful as possible to real-world surfaces, so any processing you apply risks shifting your surfaces further from the 'truth'.</span></span>

<span data-ttu-id="332ef-186">下面是一些可能有用的不同类型网格处理的示例:</span><span class="sxs-lookup"><span data-stu-id="332ef-186">Here are some examples of different types of mesh processing that you may find useful:</span></span>

* <span data-ttu-id="332ef-187">**孔洞填充**</span><span class="sxs-lookup"><span data-stu-id="332ef-187">**Hole filling**</span></span>
   * <span data-ttu-id="332ef-188">如果导致暗材料的小对象无法扫描, 它将在周围表面上留下一个孔。</span><span class="sxs-lookup"><span data-stu-id="332ef-188">If a small object made of a dark material fails to scan, it will leave a hole in the surrounding surface.</span></span>
   * <span data-ttu-id="332ef-189">孔洞会影响封闭: 在被视为不透明的实际表面, 可以通过 "通过" 洞来查看全息影像。</span><span class="sxs-lookup"><span data-stu-id="332ef-189">Holes affect occlusion: holograms can be seen 'through' a hole in a supposedly opaque real-world surface.</span></span>
   * <span data-ttu-id="332ef-190">孔洞会影响 raycasts: 如果使用 raycasts 帮助用户与表面交互, 则这些光线可能不需要通过洞。</span><span class="sxs-lookup"><span data-stu-id="332ef-190">Holes affect raycasts: if you are using raycasts to help users interact with surfaces, it may be undesirable for these rays to pass through holes.</span></span> <span data-ttu-id="332ef-191">一种缓解措施是使用多个 raycasts 的捆绑, 涵盖适当大小的区域。</span><span class="sxs-lookup"><span data-stu-id="332ef-191">One mitigation is to use a bundle of multiple raycasts covering an appropriately sized region.</span></span> <span data-ttu-id="332ef-192">这将允许你筛选 "离群离群" 结果, 这样即使一个 raycast 通过一个小孔, 聚合结果仍将有效。</span><span class="sxs-lookup"><span data-stu-id="332ef-192">This will allow you to filter 'outlier' results, so that even if one raycast passes through a small hole, the aggregate result will still be valid.</span></span> <span data-ttu-id="332ef-193">但请注意, 这种方法会产生计算成本。</span><span class="sxs-lookup"><span data-stu-id="332ef-193">However, be aware that this approach comes at a computational cost.</span></span>
   * <span data-ttu-id="332ef-194">孔洞会影响物理学冲突: 物理学模拟控制的对象可能会贯穿地面的孔, 并会丢失。</span><span class="sxs-lookup"><span data-stu-id="332ef-194">Holes affect physics collisions: an object controlled by physics simulation may drop through a hole in the floor and become lost.</span></span>

* <span data-ttu-id="332ef-198">**删除 Hallucination**</span><span class="sxs-lookup"><span data-stu-id="332ef-198">**Hallucination removal**</span></span>
   * <span data-ttu-id="332ef-199">反射、明亮灯光和移动对象可能会使小延迟 "hallucinations" 浮动在空中。</span><span class="sxs-lookup"><span data-stu-id="332ef-199">Reflections, bright lights and moving objects can leave small lingering 'hallucinations' floating in mid-air.</span></span>
   * <span data-ttu-id="332ef-200">Hallucinations 影响封闭: Hallucinations 可能会成为深色形状, 并将其移到上, 并 occluding 其他全息影像。</span><span class="sxs-lookup"><span data-stu-id="332ef-200">Hallucinations affect occlusion: hallucinations may become visible as dark shapes moving in front of and occluding other holograms.</span></span>
   * <span data-ttu-id="332ef-201">Hallucinations 影响 raycasts: 如果你使用 raycasts 帮助用户与表面交互, 则这些光线可能会碰到 hallucination, 而不是其背后的表面。</span><span class="sxs-lookup"><span data-stu-id="332ef-201">Hallucinations affect raycasts: if you are using raycasts to help users interact with surfaces, these rays could hit a hallucination instead of the surface behind it.</span></span> <span data-ttu-id="332ef-202">与孔洞一样, 一种缓解措施是使用许多 raycasts, 而不是使用单个 raycast, 但这会产生计算成本。</span><span class="sxs-lookup"><span data-stu-id="332ef-202">As with holes, one mitigation is to use many raycasts instead of a single raycast, but again this will come at a computational cost.</span></span>
   * <span data-ttu-id="332ef-203">Hallucinations 影响物理学冲突: 物理学模拟控制的对象可能会停滞在 hallucination 的位置, 并且无法在看似清晰的空间区域中移动。</span><span class="sxs-lookup"><span data-stu-id="332ef-203">Hallucinations affect physics collisions: an object controlled by physics simulation may become stuck against a hallucination and be unable to move through a seemingly clear area of space.</span></span>

* <span data-ttu-id="332ef-215">**平面查找**</span><span class="sxs-lookup"><span data-stu-id="332ef-215">**Plane finding**</span></span>
