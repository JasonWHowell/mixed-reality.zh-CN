# <a name="slider"></a>滑块

![滑块](images/UX/UX_Hero_Slider.jpg)

滑块是用户界面组件，使您可以通过将滑块移动到轨道上来持续更改值。可以通过直接或距离方向直接抓取滑块来移动挤压滑块。 滑杆适用于各种输入类型，如注视和气和运动控制器。
