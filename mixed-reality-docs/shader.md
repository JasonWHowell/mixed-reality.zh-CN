# <a name="shader"></a>着色器

![着色器](images/UX/UX_Hero_StandardShader.jpg)

由于全息对象与环境中的物理对象混合在一起，因此提供视觉提示在混合现实中非常重要。 MRTK 标准着色器提供各种类型的视觉效果，可与全息影像一起使用。 MRTK 标准底纹系统利用单个灵活的着色器，它可以实现类似于 Unity 的标准着色器的视觉对象，实现流畅的设计系统原则，并在混合现实设备上保持高性能。
